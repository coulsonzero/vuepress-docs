# cat

# less

分页查看文件内容，适合于大文件

less file.txt
1

快捷键

  • space键：下一页
  • b键：上一页
  • return键：下一行
  • y键：上一行
  • q键：退出less分页查看模式
  • =键：显示less页面详情
  • h键：显示less帮助文档
  • /键：显示less搜索模式，n键跳转

# head

显示文件的开头几行

# 显示开头3行
head -n 3 file.txt
1
2

以下会报错

head -n -3 file.txt

# tail

显示文件的结尾几行(默认后10行)

# 显示末尾2行
tail [-n  2] file.txt
tail [-n -2] file.txt
# 显示开头2行
tail [-n +2] file.txt
# 显示第10行
tail -n +10 file.txt | head -1
